## Usage

### Install from release

Download `Roveliese-*.zip` from the [latest GitHub release](https://github.com/roveliese/obsidian/releases/latest). Extract the `Roveliese` folder into your vault themes directory:

```text
<vault>/.obsidian/themes/
```

After installing, open **Settings > Appearance > Themes**, then choose **Roveliese**.

### Manual installation

Create a theme folder in your Obsidian vault:

```text
<vault>/.obsidian/themes/Roveliese
```

Download `manifest.json` and `theme.css` from this repository, then place both files inside that folder.

After installing, open **Settings > Appearance > Themes**, then choose **Roveliese**.

## Design

Roveliese centers on a quiet writing surface and a visible rose identity. Accents appear only where the interface needs emphasis: active controls, focus rings, links, tags, selections, and important states.

Markdown stays readable without carrying the brand color everywhere. Rose warmth sets the mood, while cool cyan and blue tones provide contrast for links, graph nodes, and informational callouts.

## What's Covered

- Dark and light mode
- Editor, reading view, sidebars, tabs, status bar, modals, menus, and prompts
- Markdown headings, links, tags, code blocks, blockquotes, tables, checkboxes, embeds, and highlights
- Callouts, graph view, search, backlinks, metadata, and file navigation
- Code syntax colors adapted from the Roveliese VS Code palette
